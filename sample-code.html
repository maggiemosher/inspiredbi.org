## 6. Build the Website

Build in this order. Complete each phase before moving on.

### Phase A — Foundation
1. Set up `app/layout.tsx` with Google Fonts, metadata from siteContent, Navbar, Footer
2. Build sticky `Navbar.tsx`: logo left, nav links right, mobile hamburger → Sheet drawer
3. Build minimal `Footer.tsx`: tagline, nav links, copyright

### Phase B — Home Page Sections (in order)
- **Hero**: full-viewport, large headline + 2 CTAs, subtle gradient background, no stock photos
- **About**: two-column layout, text left, credentials/stats right
- **Services**: grid of cards with icon, title, description, outcomes list. Hover: lift + border
- **Testimonials**: quote cards, auto-scrolling carousel if more than 2
- **CTA**: high-contrast band, single action

### Phase C — Inner Pages
- `/about` — expand About content, add timeline/milestones
- `/services` — full detail per service, FAQ accordion at bottom (5-8 questions)
- `/contact` — form + email link, client-side validation, Server Action on submit

### Design Rules (apply consistently)
- One primary accent color matching the brand tone, 3 colors maximum
- H1 = 3.5rem bold, H2 = 2rem semibold, body = 1rem/1.7
- `rounded-xl` for cards, `rounded-full` for buttons
- `shadow-sm` default, `shadow-md` on hover — never `shadow-xl`
- `transition-all duration-200` on interactive elements only
- Mobile-first. Test at 375px, 768px, 1280px.

## 7. Build the AI Guide

The AI Guide is a floating chat assistant on every page. It knows the site content,
greets visitors, answers questions, and guides them toward the right service.

### API Route (`app/api/chat/route.ts`)
```typescript
import { anthropic } from '@ai-sdk/anthropic';
import { streamText } from 'ai';
import { buildSystemPrompt } from '@/lib/ai-system-prompt';

export const runtime = 'edge';
export const maxDuration = 30;

export async function POST(req: Request) {
  const { messages } = await req.json();
  const result = streamText({
    model: anthropic('claude-haiku-4-5'),
    system: buildSystemPrompt(),
    messages,
    maxTokens: 600,
    temperature: 0.7,
  });
  return result.toDataStreamResponse();
}
```

### System Prompt (`lib/ai-system-prompt.ts`)
```typescript
import { siteContent } from './site-content';

export function buildSystemPrompt(): string {
  const { brand, about, services, contact } = siteContent;
  return `
You are ${brand.name}'s friendly website guide. Help visitors find exactly what
they need and decide whether to reach out.

## Your Personality
- Warm, knowledgeable, genuinely helpful — never salesy
- Answer in 2-4 sentences unless a longer answer is clearly needed
- Ask one clarifying question at a time
- If you don't know something, suggest they contact ${contact.email}
- Never make up services, prices, or credentials not listed below

## About ${brand.name}
${brand.description}
${about.body.join('\n\n')}
Credentials: ${about.credentials.join(' | ')}

## Services
${services.map(s => `**${s.title}**: ${s.description}\nOutcomes: ${s.outcomes.join(', ')}`).join('\n\n')}

## How to Guide Visitors
- First message: "Hi! I'm here to help you find what you need. What brings you here today?"
- Problem described → match to most relevant service
- Pricing questions → "Pricing varies by scope. The best next step is a free call."
- Ready to connect → guide to ${contact.email}
- Keep responses concise. One idea per message.
`.trim();
}
```

### Custom Hook (`components/ai-guide/useGuide.ts`)
```typescript
'use client';
import { useChat } from 'ai/react';
import { useState } from 'react';

export function useGuide() {
  const [isOpen, setIsOpen] = useState(false);
  const chat = useChat({
    api: '/api/chat',
    initialMessages: [{
      id: 'welcome',
      role: 'assistant',
      content: "Hi! I'm here to help you find what you need. What brings you here today?",
    }],
    onError: (err) => console.error('Guide error:', err),
  });
  return { ...chat, isOpen, open: () => setIsOpen(true), close: () => setIsOpen(false), toggle: () => setIsOpen(p => !p) };
}
```

## 10. Common Errors

| Error | Fix |
|---|---|
| `ANTHROPIC_API_KEY is not defined` | Add to `.env.local` AND Vercel env vars |
| AI doesn't stream | Ensure `export const runtime = 'edge'` in route.ts |
| `useChat` not found | `npm install ai@latest` — requires v3+ |
| Guide overlaps navbar | Panel z-50, navbar z-40 |
| Tailwind classes not applying | Check content paths in tailwind.config.ts |
